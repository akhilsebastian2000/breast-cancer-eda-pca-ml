# Breast Cancer Diagnosis using EDA, PCA, and Machine Learning

## Overview
This project analyzes breast cancer imaging features using EDA, PCA, and machine learning models.

---

## Dataset
- UCI Breast Cancer Dataset
- 30 numerical features (mean, SE, worst)

---

## Workflow
- Data Cleaning
- EDA
- PCA
- Feature Selection
- Model Training
